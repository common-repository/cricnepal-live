== Description ==
[Cricnepal](http://www.cricnepal.com/) Live Center plugin allows you to easily display Live Scores of Nepali Cricket on your site. The plugin is relatively easy to setup. 

This is the initial release and we are looking for feedback.   

For more information, check out [Cricnepal.com](http://www.cricnepal.com/).


== Installation ==

1. Upload `cricnepal-live plugin` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Appearance and click on Widget
4. Drag the Cricnepal Live widget to your sidebar
5. Put a suitable title and Save it.
